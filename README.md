# Real-Time Photo Album Website
This website is built by using **HTML, CSS, JavaScript, PHP, MySql**

## Use case
This webite allow user to upload realtime images and store there images on web.

## Setup requirements 
1) Use php version 8.2.12
2) Having Database which already presnt in repo make sure use that db.
3) For Upload images there is form which is use to upload images and display them.
